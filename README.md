# Estética Automotiva


## Descrição

Este é um site de estética automotiva que foi desenvolvido como parte de um projeto acadêmico. O site foi desenvolvido usando HTML, CSS e JavaScript.

## Tecnologias utilizadas

- HTML
- CSS

## Funcionalidades

- Página inicial com informações sobre a estética automotiva
- Página de contato com formulário de contato
- Página de portfólio com imagens dos trabalhos realizados
- Página de localização com mapa da localização da estética automotiva

## Como executar o projeto

1. Clone o repositório
2. Abra o arquivo index.html no navegador ou use o Live Server para visualizar o site.
