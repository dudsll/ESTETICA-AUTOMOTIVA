# Eduardo Garage — Estética Automotiva 🚗✨

Projeto de site institucional para uma estética automotiva fictícia, desenvolvido com foco em **aprendizado prático de desenvolvimento front-end**, organização de projeto e boas práticas de TI.

Este projeto não tem como objetivo principal a publicação em produção, mas sim servir como **base sólida para estudo, evolução técnica e portfólio**.

---

## 📌 Objetivo do Projeto

- Praticar **HTML semântico**
- Evoluir no uso de **CSS moderno**
- Aprender **organização de projeto**
- Aplicar conceitos reais de **arquitetura front-end**
- Preparar o projeto para futuras implementações em **JavaScript**
- Simular um fluxo de desenvolvimento próximo ao mercado

---

## 🧱 Estrutura do Projeto

├── index.html # Página inicial

├── style.css # Estilos globais do site
├── README.md # Documentação do projeto
└── imagens/ # Imagens utilizadas no site

## 📄 Páginas do Site

### 🏠 Home (`index.html`)

- Apresentação da marca Eduardo Garage
- Introdução aos serviços oferecidos
- Chamadas para contato e portfólio
- Localização com mapa incorporado

### 📸 Portfólio (`portfolio.html`)

- Exemplos visuais dos serviços realizados
- Demonstração de resultados (antes/depois – conceito)

### 📞 Contato (`contact-us.html`)

- Formulário para agendamento e contato
- Coleta de informações básicas do cliente e do veículo
